JavaScript

ภาษาจาวาสคริปต์ (JavaScript Programming Language) คือ ภาษาซึ่งถูกพัฒนาโดย Brenden Eich คงไม่ต้องสงสัยว่านี่คือหนึ่งในภาษาที่ได้รับความนิยมที่สุดตัวหนึ่งของโลก ซึ่ง JavaScript ถูกใช้ในการสร้างเว็บไซต์และเว็บแอปพลิเคชันเพื่อให้หน้าเว็บมีความไดนามิคและมีปฏิสัมพันธ์และการโต้ตอบกับผู้ใช้มากยิ่งขึ้น ซึ่งก็จะถูกใช้ร่วมกันกับ HTML และ CSS เป็นสามภาษาทหารเสือฝั่ง Frontend

ปัจจุบัน JavaScript สามารถทำงานได้ทั้งฝั่ง Back-end(Server Side) และ Front-end(Client Side) ซึ่งในอดีตจะทำได้เฉพาะฝั่ง Frontend ซึ่งปัจจุบันก็มีตัว Node.js ที่เข้ามาเติมเต็มให้ทำงานได้ในฝั่ง Backend แล้ว และก็ได้รับความนิยมสุด ๆ ในปัจจุบัน ในการพัฒนาด้านเว็บแอปพลิเคชัน
